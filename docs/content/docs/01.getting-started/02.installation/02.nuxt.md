### Initialize styleframe in your project

Use the CLI to initialize `styleframe` in your project. Make sure you have a `nuxt.config.ts` file in your project root.

:::code-group

```bash [pnpm]
pnpx styleframe init
```

```bash [yarn]
yarn create styleframe
```

```bash [npm]
npx styleframe init
```

```bash [bun]
bunx styleframe init
```

:::

Using the CLI command above, it will:

- Install `styleframe` as a development dependency to your project
- Add the `styleframe` module to Nuxt, if possible
- Create a new `styleframe.config.ts` file

### Import styleframe in your project

Open your `main.ts` file and import the auto-generated virtual module:


```bash [main.ts]
import 'virtual:styleframe.css';
```
