## Overview

Styleframe is offered under two licensing options to suit different needs:

- **Styleframe Community** (core library) is free and open source under the MIT License
- **Styleframe Pro** is free for development, requires a license for production
    - One Pro license covers unlimited personal and commercial projects

## License Activation

Activating your Styleframe Pro license is simple:

::steps

### Get Your License Key

After purchasing, you'll receive a license key via email. You can also find it in your [account dashboard](https://app.styleframe.dev).

### Activate Your Key

Log into your [Styleframe account](https://app.styleframe.dev) and activate your license key through the license management form.

### Set Environment Variable

Add your license key as an environment variable in your production environment:

```bash
# Add to your .env file or deployment environment
STYLEFRAME_KEY=your_license_key_here
```

::

::caution
**Important:** Never commit your license key to version control. Use environment variables and keep your key secure. Contact us if you suspect your key has been compromised.
::

### Special License Types

#### Builder License (Theme/Plugin Developers)

If you're building **themes, plugins, templates, or tools** that you want to sell or distribute with Styleframe Pro embedded, you need a **Builder License**.

A Builder License allows you to:
- Distribute products with Styleframe Pro embedded
- Sell themes, templates, and starter kits
- Create reusable components for redistribution
- Build tools and libraries on top of Styleframe Pro

::note{icon="i-ph-storefront"}
**Need a Builder License?** Builder Licenses are sold separately and renewed annually. [Contact us](mailto:support@styleframe.dev) to discuss your use case and pricing.
::

## What You Can Do with Styleframe Pro

### Allowed Uses

With a valid Styleframe Pro license, you and your team members can:

#### Development & Collaboration
- Install and use Styleframe Pro on team computers
- Create backup copies for security and continuity
- Modify Styleframe Pro components for your projects
- Share copies and modifications with other team members

#### Project Integration
- Embed Styleframe Pro in websites and applications
- Build and deploy commercial projects
- Create client websites and applications
- Develop SaaS products and web applications
- Publish projects that include Styleframe Pro

#### End User Rights
- Grant end users permission to use, copy, and run your projects
- Distribute projects incorporating Styleframe Pro
- Sell products built with Styleframe Pro

### Restrictions

To protect the value of Styleframe Pro, the following are **not allowed**:

#### Distribution Limitations
- **Cannot** distribute standalone copies of Styleframe Pro to non-team members
- **Cannot** share your license key with people outside your organization
- **Cannot** authorize third parties to modify or embed Styleframe Pro directly

#### License Bypassing
- **Cannot** bypass licensing by distributing Styleframe Pro in projects where end users need their own license (unless you have a Builder License - see below)
- **Cannot** continue using Styleframe Pro after license expiration or termination
- **Cannot** host projects using Styleframe Pro without an active license

## Frequently Asked Questions

::accordion

:::accordion-item{label="Do I need a license for every project?" icon="i-lucide-circle-help"}
No! One Styleframe Pro license covers unlimited projects. Build as many sites and applications as you want.
:::

:::accordion-item{label="Can I use Styleframe Pro while evaluating it?" icon="i-lucide-circle-help"}
Yes! Styleframe Pro is completely free in development mode. Test all features locally before purchasing a production license.
:::

:::accordion-item{label="Can I use Styleframe Pro in open source projects?" icon="i-lucide-circle-help"}
Yes! Your license covers open source projects. However, each user will need their own Styleframe Pro license. If your open source project is a redistributable theme, plugin, or tool, you may need a Builder License.
:::

:::accordion-item{label="How is the license verified?" icon="i-lucide-circle-help"}
Styleframe Pro checks for a valid `STYLEFRAME_KEY` environment variable in production. Development environments (localhost, common development ports) don't require a license key.
:::

:::accordion-item{label="What if I forget to set the environment variable?" icon="i-lucide-circle-help"}
Styleframe Pro may display watermarks or console warnings in production when running without a valid license key.
:::

:::accordion-item{label="Can I transfer my license to a different organization?" icon="i-lucide-circle-help"}
License transfers are evaluated on a case-by-case basis. [Contact support](mailto:support@styleframe.dev) to discuss your situation.
:::

:::accordion-item{label="Can I build client websites with Styleframe Pro?" icon="i-lucide-circle-help"}
Yes! Your license covers unlimited client projects. Each site you build for clients is covered under your single license.
:::

:::accordion-item{label="Can I sell SaaS applications built with Styleframe Pro?" icon="i-lucide-circle-help"}
Yes! You can build and sell commercial applications. Your license covers the use of Styleframe Pro in your SaaS product.
:::

:::accordion-item{label="Can I sell WordPress themes or UI kits using Styleframe Pro?" icon="i-lucide-circle-help"}
If you want to distribute products that include Styleframe Pro (themes, templates, starter kits), you need a Builder License. [Contact us](mailto:support@styleframe.dev) for details.
:::

:::accordion-item{label="What about my clients' licenses?" icon="i-lucide-circle-help"}
When you build a website for a client using Styleframe Pro:
- Your license covers the development and deployment
- The client doesn't need their own license if you're just delivering a finished website
- If the client will be modifying or embedding Styleframe Pro themselves, they need their own license
:::
